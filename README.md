# Colabor - Módulo de Anotaciones Colaborativas para Dolibarr

**Colabor** es un módulo diseñado para potenciar la comunicación interna dentro de Dolibarr. Permite añadir notas y anotaciones enriquecidas en las fichas principales del sistema, facilitando el seguimiento y la colaboración entre miembros del equipo.

## 🚀 Características principales
* **Anotaciones en tiempo real:** Agrega notas en Terceros, Productos, Presupuestos, Pedidos, Facturas y Proyectos.
* **Menciones (@):** Etiqueta a otros usuarios del sistema para notificarlos o referenciarlos en tus notas.
* **Interfaz Enriquecida:** Editor de texto integrado para dar formato a tus anotaciones.
* **Centralización:** Mantén todo el historial de comunicación en un solo lugar dentro de cada objeto de Dolibarr.

## 🛠️ Instalación
1.  Descarga el archivo ZIP del repositorio o clónalo:
    ```bash
    git clone git@github.com:MACHFREE/colabor.git
    ```
2.  Copia la carpeta `colabor` dentro del directorio `htdocs/custom/` de tu instalación de Dolibarr.
3.  Ve a **Configuración -> Módulos**, busca "Colabor" y actívalo.

## 📋 Requisitos
* **Dolibarr:** v18 o superior (Probado hasta v22).
* **PHP:** 7.4 o superior.
